PESE Lab Activities

Week 1 January 15, 2025

Introduction to Physical Computing

In our first PESE lab session, we explored the fundamentals of physical computing and its applications in everyday life. The session began with an introduction to various sensors and microcontrollers that form the backbone of modern IoT devices.

During the hands-on portion, we assembled basic circuits using Arduino boards and learned how to write simple programs to control LED lights and read sensor data. This gave us a practical understanding of how software interacts with hardware components.

Key concepts covered:

  • Basics of electrical circuits
  • Introduction to Arduino programming
  • Working with digital inputs and outputs
  • Understanding sensor data collection
Week 2 January 22, 2025

Data Visualization Techniques

This week's PESE lab focused on techniques for visualizing data collected from various sensors. We learned how to process raw sensor data and transform it into meaningful visual representations that can be easily interpreted.

The session featured a workshop on using Processing and Python libraries such as Matplotlib and Seaborn to create interactive visualizations. We worked with real-time data streams from temperature and motion sensors to create dynamic graphs and charts.

By the end of the session, each team had created a dashboard that displayed various environmental metrics collected from our lab environment. This exercise demonstrated the importance of effective data visualization in interpreting complex datasets.

Projects completed:

  • Real-time temperature monitoring system
  • Motion pattern visualization
  • Environmental data dashboard
Week 3 January 29, 2025

Interactive Installation Design

Week 3 of our PESE lab introduced us to the principles of designing interactive installations that respond to human presence and behavior. We discussed how physical computing can be used to create engaging experiences in public spaces, museums, and art galleries.

The practical component involved working in small groups to conceptualize and prototype a small-scale interactive installation. Using motion sensors, LEDs, speakers, and various microcontrollers, each group created a unique interactive experience that responded to user proximity, movement, or touch.

Our instructor provided valuable feedback on each prototype, highlighting considerations for scalability, user experience, and technical robustness. We also discussed ethical considerations in designing installations that collect data from participants.

Highlights from the session:

  • Principles of interaction design
  • Sensor fusion techniques
  • Prototyping interactive experiences
  • User testing methodologies
